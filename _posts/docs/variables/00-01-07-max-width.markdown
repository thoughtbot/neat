{% highlight sass %}
$max-width
  type: number (positive, unit)
  default: em(1088)
{% endhighlight %}
- Sets the max-width property of the element that includes `outer-container()`.

#### Usage Notes

- To learn more about `em()` see [Bourbon docs](http://bourbon.io/docs/#px-to-em).
